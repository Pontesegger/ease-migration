Eclipse Advanced Scripting Environment (EASE)

EASE provides scripting support, allowing to interact with the IDE using scripts in various languages.

Scripts do have access to their native language features, eg python scripts do have access to all available python libraries. Further scripts do have access to the Java runtime, allowing to access and manipulate Java objects. Therefore scripts operate on the same level as normal Eclipse plug-ins.

Currently following script engines are supported:

  • JavaScript (Rhino)
  • JavaScript (Nashorn)
  • Jython
  • Python via Py4J
  • Ruby
  • Groovy
